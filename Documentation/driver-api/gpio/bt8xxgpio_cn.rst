这是一个自制的低成本基于BT8xx的PCI GPIO卡（bt8xxgpio）的驱动程序。

对于高级文档，请参见 https://bues.ch/cms/unmaintained/btgpio.html

可以使用普通的Brooktree bt848、bt849、bt878或bt879模拟电视调谐卡构建一个通用的24端口数字PCI GPIO卡。这些Brooktree芯片被用在老旧的Hauppauge WinTV PCI模拟电视卡中。你可以在网络上轻松地以低价找到它们。
BT8xx芯片拥有24个数字GPIO端口。
这些端口可通过SMD芯片封装上的24个引脚访问。
如何物理访问GPIO引脚
=====================

有几种方法可以访问这些引脚。一种方法是将整个芯片拆焊并放置在一个自定义的PCI板上，或者只将每个单独的GPIO引脚拆焊并焊接在一些细小的线上。由于芯片封装非常小，因此无论哪种情况都需要一定的高级焊接技巧。
