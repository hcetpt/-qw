===========================
mac80211 子系统（基础）
===========================

在实现 mac80211 驱动程序时，您应该阅读并理解本书这部分包含的信息。在某些章节中，提到了高级用法，如果不需要这些内容，则可以跳过。本书的这一部分仅涵盖工作站模式和监控模式的功能，实现其他模式所需的信息将在本书的第二部分中介绍。
