数字电视（DVB）设备
------------------------

数字电视设备由多个不同的驱动程序实现：

- **桥接驱动程序**：负责与其它设备连接的总线（如 PCI、USB、SPI）进行通信，绑定到其它驱动程序，并实现数字解复用逻辑（可在软件或硬件中实现）；

- **前端驱动程序**：通常被实现为两个独立的驱动程序：

  - **调谐器驱动程序**：实现了控制硬件中用于调谐至数字电视转发器或物理频道部分的逻辑。调谐器的输出通常是基带信号或中频（IF）信号；

  - **解调器驱动程序**（也称为“解调”）：实现了控制数字电视解码硬件的逻辑。解调器的输出是一组数字流，其中通常使用 MPEG 传输流复用多路音频、视频和数据信道。_f1_

在大多数硬件上，前端驱动程序通过 I2C 总线与桥接驱动程序通信。
.. _f1: 一些标准使用 TCP/IP 进行数据复用，例如已被废弃不再使用的 DVB-H 标准以及当前 ATSC 3.0 的提案。目前，DVB 子系统并未实现这些标准。
