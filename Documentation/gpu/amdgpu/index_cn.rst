==========================
drm/amdgpu AMDgpu驱动程序
==========================

drm/amdgpu驱动程序支持基于Graphics Core Next (GCN)、Radeon DNA (RDNA) 和 Compute DNA (CDNA) 架构的所有AMD Radeon GPU。
.. toctree::

   module-parameters
   driver-core
   display/index
   flashing
   xgmi
   ras
   thermal

为了更好地理解上述目录结构，以下是对各部分的简要说明：

- **module-parameters**：模块参数
- **driver-core**：驱动核心
- **display/index**：显示
- **flashing**：刷新
- **xgmi**：XGMI（交叉连接GPU互连）
- **ras**：可靠性、可用性和可服务性
- **thermal**：热管理
- **driver-misc**：驱动杂项
- **debugging**：调试
- **amdgpu-glossary**：术语表

这些部分详细介绍了AMDgpu驱动程序的各个方面。
