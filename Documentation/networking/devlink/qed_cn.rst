```SPDX 许可声明标识符: GPL-2.0
===================
qed devlink 支持
===================

本文档描述了由 ``qed`` 核心设备驱动程序实现的 devlink 功能。
参数
==========

``qed`` 驱动实现了以下驱动特有的参数：

.. 列表表格:: 实现的驱动特有的参数
   :宽度: 5 5 5 85

   * - 名称
     - 类型
     - 模式
     - 描述
   * - ``iwarp_cmt``
     - 布尔值
     - 运行时
     - 为 100G 设备启用 iWARP 功能。请注意，这会影响第 2 层性能，因此默认情况下不启用。
```
