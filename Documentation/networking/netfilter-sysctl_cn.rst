=========================
Netfilter Sysfs 变量
=========================

/proc/sys/net/netfilter/* 变量:
====================================

nf_log_all_netns - 布尔值
	- 0 - 禁用（默认）
	- 非 0 - 启用

	默认情况下，只有 init_net 命名空间可以将数据包记录到内核日志中，使用 LOG 目标；这旨在防止容器淹没主机的内核日志。如果启用，此目标在其他网络命名空间中也能工作。此变量仅在 init_net 中可访问
