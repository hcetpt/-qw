rv-mon
======
-----------------------
列出可用的监视器
-----------------------

:手册章节: 1

概要
====

**rv mon** [*-h*] **监视器名称** [*-h*] [*监视器选项*]

描述
====

**rv mon** 命令运行名为 *监视器名称* 的监视器。每个监视器都有一组自己的选项。使用 **rv list** 命令可以显示所有可用的监视器。
选项
====

**-h**, **--help**

        打印帮助菜单
可用监视器
==================

**rv** 工具为一组监视器提供接口。使用 **rv list** 命令来列出所有可用的监视器。每个监视器都有其自己的一套选项。查看 **rv-mon**-*监视器名称* 手册页以获取有关每个特定监视器的详细信息。另外，运行 **rv mon** **监视器名称** **-h** 可以显示带有可用选项的帮助菜单。
参见
====

**rv**(1), **rv-mon**(1)
