=======
rv-list
=======
-----------------------
列出可用监视器
-----------------------

:手册章节: 1

概要
========

**rv list** [*选项*]

描述
===========

**rv list** 命令打印所有可用的监视器。这些监视器可以通过 **rv mon** 命令启用。
选项
=======

**-h**, **--help**

        打印帮助菜单
参考
========

**rv**\(1\), **rv-mon**\(1\)

Linux 内核 *RV* 文档:
<https://www.kernel.org/doc/html/latest/trace/rv/index.html>

======
