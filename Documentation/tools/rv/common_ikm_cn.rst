**-h**, **--help**

        打印监视器的选项和可用反应器列表

**-r**, **--reactor** *reactor*

        启用*reactor*。参见**-h**以获取可用反应器的列表。
        
**-s**, **--self**

        当进行追踪（**-t**）时，也打印在**rv**命令自身期间发生的事件。如果**rv**命令自身产生过多的事件，
        工具可能会忙于仅仅处理它自己的事件。

**-t**, **--trace**

        追踪监视器的事件和错误
        
**-v**, **--verbose**

        打印调试信息
