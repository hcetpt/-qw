rv

--------------------
运行时验证
--------------------

:手册章节: 1

概要
========

**rv** *命令* [*选项*]

描述
===========

运行时验证（**RV**）是一种轻量级（但严谨的）形式化验证方法，采用实用的方法来处理复杂系统。它不依赖于系统的细粒度模型（例如，在指令级别上的重新实现），而是通过分析系统实际执行的轨迹，并将其与系统行为的形式化规范进行比较来进行工作。
**rv** 工具提供了一系列运行时验证（rv）监视器的接口。

命令
========

**list**

        列出所有可用的监视器
**mon**

        运行监视器
选项
=======

**-h**, **--help**

        显示帮助文本
对于其他选项，请参阅相应命令的手册页。
