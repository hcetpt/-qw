概述
======
**rtla** *命令* [*选项*]

描述
===========
**rtla** 是一个元工具，它包含了一系列命令，旨在分析Linux的实时特性。与将Linux作为一个黑盒进行测试不同，**rtla** 利用内核跟踪功能来提供关于这些特性和意外结果的根本原因的精确信息。

命令
========
**osnoise**

        提供操作系统噪声（osnoise）的信息。
**timerlat**

        测量IRQ和线程定时器延迟。

选项
=======
**-h**, **--help**

        显示帮助文本。
对于其他选项，请参阅相应命令的手册页。
