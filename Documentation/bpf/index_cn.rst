BPF 文档
========

此目录包含关于 BPF（Berkeley Packet Filter）功能的文档，重点放在扩展版 BPF（eBPF）上。
内核侧的文档仍在开发中。
Cilium 项目还维护了一个《BPF 和 XDP 参考指南》_ ，深入技术层面介绍了 BPF 架构。
