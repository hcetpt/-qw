========================
数字电视解复用 close()
========================

名称
----

数字电视解复用 close()

概览
--------

.. c:函数:: int close(int fd)

参数
---------

``fd``
  由先前的 :c:func:`open()` 调用返回的文件描述符
描述
-----------

此系统调用会停用并释放通过先前 :c:func:`open()` 调用分配的过滤器。
返回值
------------

成功时返回 0
出错时返回 -1，并且设置 ``errno`` 变量为适当的错误代码
通用错误代码在 :ref:`通用错误代码 <gen-errors>` 章节中进行了描述
