***********************************************
数字电视前端旧版 API（又称 DVBv3）
***********************************************

使用此 API 已被废弃，因为它不支持所有的数字电视标准，不能提供良好的统计测量，并且提供的信息不完整。保留此 API 仅为了支持旧版应用程序。
