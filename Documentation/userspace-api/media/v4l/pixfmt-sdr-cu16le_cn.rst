****************************
V4L2_SDR_FMT_CU16LE ('CU16')
****************************

小端字节序的复数无符号16位IQ采样

描述
===========

此格式包含一系列复数采样。每个复数由两部分组成，称为同相（I）和正交（Q）。I和Q均表示为16位无符号小端字节序数字。I值在前，Q值随后。
**字节顺序。**
每个单元格是一个字节
.. flat-table::
    :header-rows:  0
    :stub-columns: 0

    * - 起始 + 0:
      - I'\ :sub:`0[7:0]`
      - I'\ :sub:`0[15:8]`
    * - 起始 + 2:
      - Q'\ :sub:`0[7:0]`
      - Q'\ :sub:`0[15:8]`
