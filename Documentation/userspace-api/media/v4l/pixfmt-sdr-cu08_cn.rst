*************************
V4L2_SDR_FMT_CU8 ('CU08')
*************************

复数无符号 8 位 IQ 样本

描述
===========

此格式包含一系列复数样本。每个复数由两部分组成，称为同相和正交（IQ）。I 和 Q 都表示为一个 8 位无符号数。I 值在前，Q 值随后。
**字节顺序：**
每个单元为一个字节

.. flat-table::
    :header-rows:  0
    :stub-columns: 0

    * - 起始 + 0:
      - I\ :sub:`0`
    * - 起始 + 1:
      - Q\ :sub:`0`
