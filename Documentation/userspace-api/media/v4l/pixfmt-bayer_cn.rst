*****************
原始 Bayer 格式
*****************

描述
===========

原始 Bayer 格式用于图像传感器在对图像进行任何处理之前。这些格式包含绿色、红色和蓝色组件，交替的行中包含红色和绿色像素以及蓝色和绿色像素，排列顺序不同。请参阅 `Wikipedia 上关于 Bayer 滤镜的文章 <https://en.wikipedia.org/wiki/Bayer_filter>`__。
