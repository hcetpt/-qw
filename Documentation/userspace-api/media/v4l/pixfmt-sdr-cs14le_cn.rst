****************************
V4L2_SDR_FMT_CS14LE ('CS14')
****************************

复数有符号 14 位小端序 IQ 样本

描述
===========

此格式包含一系列复数样本。每个复数由两部分组成，称为同相分量和正交分量（IQ）。I 和 Q 都表示为 14 位有符号小端序数字。I 值在前，Q 值随后。14 位值存储在 16 位空间中，未使用的高位用 0 填充。
**字节顺序。**
每个单元是一个字节
.. flat-table::
    :header-rows:  0
    :stub-columns: 0

    * - 起始 + 0:
      - I'\ :sub:`0[7:0]`
      - I'\ :sub:`0[13:8]`
    * - 起始 + 2:
      - Q'\ :sub:`0[7:0]`
      - Q'\ :sub:`0[13:8]`
