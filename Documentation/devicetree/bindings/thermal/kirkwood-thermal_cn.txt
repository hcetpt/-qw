*Kirkwood 温度监控*

本版本适用于 Kirkwood 88F8262 与 88F6283 系统级芯片 (SoC)。其他 Kirkwood 芯片并未内置温度传感器。
所需属性：
- compatible: "marvell,kirkwood-thermal"
- reg: 温度寄存器的地址范围

示例：

``` 
	thermal@10078 {
		compatible = "marvell,kirkwood-thermal";
		reg = <0x10078 0x4>;
	};
```
这段配置表示在地址 0x10078 处有一个长度为 0x4（即 4 字节）的温度寄存器区域，它与 "marvell,kirkwood-thermal" 兼容。
