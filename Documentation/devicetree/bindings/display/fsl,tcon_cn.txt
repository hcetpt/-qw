Freescale TCON 驱动的设备树绑定

所需属性：
- compatible：      应该是下列之一
    * "fsl,vf610-tcon"
- reg：            TCON 寄存器集的地址和长度
- clocks：         根据通用时钟绑定：指向 TCON IPG 时钟的句柄
- clock-names：     根据通用时钟绑定：应为 "ipg"

示例：
``` 
timing-controller@4003d000 {
	compatible = "fsl,vf610-tcon";
	reg = <0x4003d000 0x1000>;
	clocks = <&clks VF610_CLK_TCON0>;
	clock-names = "ipg";
};
```
这段代码描述了一个在地址 `0x4003d000` 开始，长度为 `0x1000` 的 TCON 控制器节点。它指定了与 Freescale VF610 TCON 兼容，并且指定了其使用的时钟为 TCON IPG 时钟。
