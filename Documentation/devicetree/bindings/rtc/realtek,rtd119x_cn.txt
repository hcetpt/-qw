Realtek RTD129x 实时时钟
===============================

必需属性：
- compatible：应为 "realtek,rtd1295-rtc"
- reg        ：指定物理基地址和大小
- clocks     ：指定时钟门控

示例：

	rtc@9801b600 {
		compatible = "realtek,rtd1295-clk";
		reg = <0x9801b600 0x100>;
		clocks = <&clkc RTD1295_CLK_EN_MISC_RTC>;
	};

请注意，示例中的 `compatible` 的值应更正为 `"realtek,rtd1295-rtc"` 以符合文档的要求。以下是修正后的示例：

	rtc@9801b600 {
		compatible = "realtek,rtd1295-rtc";
		reg = <0x9801b600 0x100>;
		clocks = <&clkc RTD1295_CLK_EN_MISC_RTC>;
	};
