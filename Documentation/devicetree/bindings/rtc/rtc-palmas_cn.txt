帕尔马斯 (Palmas) RTC 控制器绑定

必需属性：
- compatible:
  - "ti,palmas-rtc" 对于 RTC 控制器的帕尔马斯系列
- interrupts: 设备上 RTC 子模块的中断编号
可选属性：

- ti,backup-battery-chargeable: 帕尔马斯系列设备如 TPS65913 或
	TPS80036 支持备用电池为 RTC 供电，当主电池被移除或处于极低功耗状态时。备用电池可以是可充电的或不可充电的。此标志用于指示电池是否可充电。如果电池可充电，则驱动程序可以启用充电功能
- ti,backup-battery-charge-high-current: 启用备用电池的大电流充电。设备支持 < 100uA 和 > 100uA 的充电电流
大电流充电将大于 100uA。如果缺少此属性，则将以较小的电流（即 < 100uA）给电池充电
示例：
```
palmas: tps65913@58 {
	..
palmas_rtc: rtc {
			compatible = "ti,palmas-rtc";
			interrupt-parent = <&palmas>;
			interrupts = <8 0>;
			ti,backup-battery-chargeable;
			ti,backup-battery-charge-high-current;
		};
		..
};
```
这段文本描述了如何在硬件配置文件中定义一个与帕尔马斯 (Palmas) 系列芯片中的实时时钟 (RTC) 控制器相关的设备节点，并指定了该节点所需的属性以及一些可选属性。
