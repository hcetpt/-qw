针对MediaTek PMIC（电源管理集成电路）的RTC（实时时钟）的设备树绑定

MediaTek PMIC 基础的 RTC 是一个独立的功能，它作为 Multi-Function Device (MFD) 的一种类型。RTC 可以通过 PMIC 包装器总线进行配置和设置，PMIC 包装器总线是一种与同一 PMIC 上其他功能共享的公共资源。
对于 MediaTek PMIC MFD 绑定，请参阅：
../mfd/mt6397.txt

对于 MediaTek PMIC 包装器总线绑定，请参阅：
../soc/mediatek/pwrap.txt

必需属性：
- compatible: 应该是以下选项之一
       "mediatek,mt6323-rtc": 适用于 MT6323 PMIC
       "mediatek,mt6358-rtc": 适用于 MT6358 PMIC
       "mediatek,mt6366-rtc", "mediatek,mt6358-rtc": 适用于 MT6366 PMIC
       "mediatek,mt6397-rtc": 适用于 MT6397 PMIC

示例：

       pmic {
               compatible = "mediatek,mt6323";

               ..
rtc {
                       compatible = "mediatek,mt6323-rtc";
               };
       };

此示例展示了如何在设备树中声明一个基于MediaTek PMIC的RTC设备。在这个例子中，PMIC被标识为MT6323型号，并且其中包含了RTC组件。
