* Dallas DS1390    SPI 串行实时钟

必需属性：
- compatible: 应包含 "dallas,ds1390"
- reg: 芯片的 SPI 地址

可选属性：
- trickle-resistor-ohms : 为涓流充电器选择的电阻值
    对于 ds1390 可用的值为 250、2000、4000
    如果要启用涓流充电器，则应给出该值
- trickle-diode-disable : 不使用内部的涓流充电二极管
    如果需要禁用内部的涓流充电二极管，则应给出此选项

示例：
```
ds1390: rtc@0 {
    compatible = "dallas,ds1390";
    trickle-resistor-ohms = <250>;
    reg = <0>;
};
```
