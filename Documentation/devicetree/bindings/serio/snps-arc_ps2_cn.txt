* ARC PS/2 驱动程序：在某些 ARC FPGA 和 nSIM OSCI 模型中使用的 PS/2 模块。

所需属性：
- compatible（兼容性）		: "snps,arc_ps2"
- reg（寄存器）			: 寄存器的偏移量和长度（始终为 0x14）
- interrupts（中断）		: 中断
- interrupt-names（中断名称）	: 中断名称，必须为 "arc_ps2_irq"

示例：

serio@c9000400 {
	compatible = "snps,arc_ps2";
	reg = <0xc9000400 0x14>;
	interrupts = <13>;
	interrupt-names = "arc_ps2_irq";
}
