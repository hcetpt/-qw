* 适用于 TI TPS61050/61052 升压转换器的设备树绑定

TPS61050/TPS61052 是一款高功率“白色 LED 驱动器”。该
设备提供 LED、GPIO 和稳压器功能。
必需属性：
- compatible:       “ti,tps61050” 或 “ti,tps61052”
- reg:             指定 I2C 从地址

可选子节点：

此子节点选择芯片的工作模式
最多只能有一个可用子节点
- regulator: 存在该子节点使芯片处于稳压器模式
参见 ../regulator/regulator.yaml

- led: 存在这个子节点使芯片处于 LED 模式
可选属性：
    - function : 参见 ../leds/common.txt
    - color    : 参见 ../leds/common.txt
    - label    : 参见 ../leds/common.txt
                  （已废弃）

示例（仅 GPIO 操作）:

```yaml
i2c0 {
	tps61052@33 {
		compatible = "ti,tps61052";
		reg = <0x33>;
	};
};
```

示例（GPIO + 稳压器操作）:

```yaml
i2c0 {
	tps61052@33 {
		compatible = "ti,tps61052";
		reg = <0x33>;

		regulator {
			regulator-min-microvolt = <5000000>;
			regulator-max-microvolt = <5000000>;
			regulator-always-on;
		};
	};
};
```

示例（GPIO + LED 操作）:

```yaml
#include <dt-bindings/leds/common.h>

i2c0 {
	tps61052@33 {
		compatible = "ti,tps61052";
		reg = <0x33>;

		led {
			color = <LED_COLOR_ID_WHITE>;
		};
	};
};
```
