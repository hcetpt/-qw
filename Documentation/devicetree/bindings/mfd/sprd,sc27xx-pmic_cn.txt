展讯 SC27xx 系列电源管理集成电路 (PMIC)

展讯 SC27xx 系列 PMIC 包含 SC2720、SC2721、SC2723、SC2730 和 SC2731。展讯 SC27xx 系列的 PMIC 集成了所有移动手机电源管理、音频编解码器、电池管理和用户界面支持功能于单一芯片中。它具有以下六个主要功能模块：
- DCDC：用于支持 CPU 和内存
- LDO：用于满足内部和外部需求
- 电池管理系统，例如充电器、电量计
- 音频编解码器
- 用户界面功能，如指示灯、闪光 LED 等
- 芯片级接口，例如电源开关控制、实时时钟 (RTC) 和 Type-C 接口等

所需属性：
- 兼容性：应为以下选项之一：
   "sprd,sc2720"
   "sprd,sc2721"
   "sprd,sc2723"
   "sprd,sc2730"
   "sprd,sc2731"
- reg：设备芯片选择的地址，应为 0
- spi-max-frequency：通常设置为 26000000
- interrupts：设备连接到的中断线
- interrupt-controller：将设备节点标记为中断控制器
这段代码描述了一个电源管理集成电路 (PMIC) 设备在设备树中的配置，设备树是一种用于描述硬件组件及其特性的数据结构，在嵌入式系统中非常常见。下面是这段配置的中文翻译：

- `#interrupt-cells`：描述一个 PMIC 中断（IRQ）所需的单元格数量，必须是 2。
- `#address-cells`：子设备偏移地址所需的单元格数量，必须是 1。
- `#size-cells`：子设备大小所需的单元格数量，必须是 0。

示例：
```plaintext
pmic@0 { 
	compatible = "sprd,sc2731"; // 兼容性字符串，表示这个节点代表的是展讯 SC2731 PMIC
	reg = <0>; // 寄存器基址，这里表示基址为 0
	spi-max-frequency = <26000000>; // SPI 最大频率为 26MHz
	interrupts = <GIC_SPI 31 IRQ_TYPE_LEVEL_HIGH>; // 中断信息，表示 GIC SPI 中断 31，中断类型为高电平有效
	interrupt-controller; // 表示这是一个中断控制器
	#interrupt-cells = <2>; // 描述中断所需的单元格数量为 2
	#address-cells = <1>; // 子设备地址偏移所需的单元格数量为 1
	#size-cells = <0>; // 子设备大小所需的单元格数量为 0
};
```

这段配置指定了一个名为 `pmic@0` 的设备节点，它与展讯 SC2731 PMIC 相兼容，并详细定义了其寄存器、SPI 最大频率、中断控制器特性以及一些必要的计数单元。
