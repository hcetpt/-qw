KEYMILE qrio 板载控制 CPLD

qrio 是一个多功能设备，它基于 kmp204x 设计来控制 KEYMILE 的板卡。
它包含了一个复位控制器、看门狗定时器、LEDs 和两个具有中断请求（IRQ）能力的 GPIO 模块。
必需的属性：
- 兼容性: "keymile,qriox"
- reg: 在父级本地总线上的访问（芯片选择、芯片选择内的偏移量、大小）

示例：

``` 
board-control@1,0 { 
    compatible = "keymile,qriox"; 
    reg = <1 0 0x80>; 
};
```

这段示例描述了如何在一个设备树中定义一个名为 `board-control` 的设备，并位于地址 `1,0` 处。这个设备与 `"keymile,qriox"` 兼容，表示它是一个 KEYMILE 的 qrio 控制器。`reg` 属性指定了该设备在父级本地总线上的访问方式，这里使用的是芯片选择 1，在该芯片选择中的偏移量为 0，访问大小为 0x80。
