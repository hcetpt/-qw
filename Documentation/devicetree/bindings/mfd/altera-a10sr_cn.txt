* Altera Arria10 开发套件系统资源芯片

所需父设备属性：
- compatible		: "altr,a10sr"
- spi-max-frequency	: 最大 SPI 频率
- reg			: Arria10 系统资源芯片的 SPI 芯片选择地址
- interrupts		: 设备所连接的中断线
- interrupt-controller	: 标记该设备节点为一个中断控制器
- #interrupt-cells	: 描述一个 IRQ 所需的单元格数量，应为 2
第一个单元格是 IRQ 编号
第二个单元格是标志，编码为来自 ../interrupt-controller/interrupts.txt 中的触发掩码
A10SR 包含以下子设备：

设备                   描述
------                   ----------
a10sr_gpio              GPIO 控制器
a10sr_rst               重置控制器

Arria10 GPIO
所需属性：
- compatible        : 应为 "altr,a10sr-gpio"
- gpio-controller   : 标记该设备节点为一个 GPIO 控制器
- #gpio-cells       : 应为两个。第一个单元格是针脚编号，第二个单元格用于指定标志
更多信息请参见 ../gpio/gpio.txt
Arria10 外设 PHY 重置
所需属性：
- compatible        : 应为 "altr,a10sr-reset"
- #reset-cells      : 应为一个
示例：

        resource-manager@0 {
		兼容性 = "altr,a10sr";
		注册器 = <0>;
		spi-最大频率 = <100000>;
		中断父设备 = <&portb>;
		中断 = <5 IRQ_TYPE_LEVEL_LOW>;
		中断控制器;
		#中断单元格 = <2>;

		a10sr_gpio: gpio控制器 {
			兼容性 = "altr,a10sr-gpio";
			gpio控制器;
			#gpio单元格 = <2>;
		};

		a10sr_rst: 重置控制器 {
			兼容性 = "altr,a10sr-reset";
			#重置单元格 = <1>;
		};
	};

请注意，上述翻译将一些专业术语直接进行了翻译，而在实际硬件文档或代码中可能会使用行业内的标准术语。例如，“注册器”可能更常被翻译为“地址空间”，“spi-最大频率”通常被描述为“SPI最大频率”，“中断控制器”和“gpio控制器”等是行业内常用的专业术语，一般不进行翻译。因此，这段翻译更适合理解为意译版本。
