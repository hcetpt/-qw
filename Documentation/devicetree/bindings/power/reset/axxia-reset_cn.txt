Axxia 重启驱动程序

此驱动程序可以对 Axxia 系统级芯片 (SoC) 进行复位。它使用 syscon 模块中的寄存器来发起芯片复位。
必需的属性：
  -compatible: "lsi,axm55xx-reset"
  -syscon: 对 syscon 节点的 phandle 引用
示例：

```
syscon: syscon@2010030000 { 
	compatible = "lsi,axxia-syscon", "syscon";
	reg = <0x20 0x10030000 0 0x2000>;
};

reset: reset@2010031000 { 
	compatible = "lsi,axm55xx-reset";
	syscon = <&syscon>;
};
```

以上内容描述了一个用于 Axxia SoC 的复位驱动程序，它可以利用 syscon 块中的寄存器来执行芯片复位操作。在 Device Tree 中，定义了必要的属性和示例配置。
