Microsemi Ocelot 重置控制器

DEVCPU_GCB:CHIP_REGS 中有一个名为 SOFT_RST 的寄存器，可以用来重置 SoC 核心。
这些重置寄存器同时存在于 MSCC vcoreiii MIPS 和 microchip Sparx5 armv8 SoC 中。
所需属性：

- compatible: "mscc,ocelot-chip-reset", "mscc,luton-chip-reset",
  "mscc,jaguar2-chip-reset" 或 "microchip,sparx5-chip-reset"

示例：
```plaintext
reset@1070008 {
	compatible = "mscc,ocelot-chip-reset";
	reg = <0x1070008 0x4>;
};
```
这段示例代码定义了一个位于地址 `0x1070008` 处的重置控制器，并且指定了其兼容性为 "mscc,ocelot-chip-reset"。
