针对德州仪器 (Texas Instruments) DA8xx 主外设优先级驱动的设备树绑定

DA8XX 系统芯片（SoC）具有一组寄存器，可以用来改变被分类为主设备的所有外设的优先级。

文档：
OMAP-L138 (DA850) - http://www.ti.com/lit/ug/spruh82c/spruh82c.pdf

必需的属性：

- compatible:         "ti,da850-mstpri" —— 对于基于 da850 的板子
- reg:                DA850 主外设优先级寄存器的偏移和长度

下面展示了 da850-lcdk 的示例：
``` 
mstpri {
    compatible = "ti,da850-mstpri";
    reg = <0x14110 0x0c>;
};
```
