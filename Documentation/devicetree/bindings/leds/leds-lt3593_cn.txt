Linear Technologies LT3593 LED 控制器的绑定说明：

必需属性：
- compatible：应设置为 "lltc,lt3593"
- lltc,ctrl-gpios：指向与芯片的 'CTRL' 引脚相连的 GPIO 的句柄

硬件仅支持一个 LED。此 LED 的属性在设备节点中的子节点中进行配置。

可选子节点属性：
- function：参见 Documentation/devicetree/bindings/leds/common.txt
- color：参见 Documentation/devicetree/bindings/leds/common.txt
- label：LED 的标签。如果没有给出，则 LED 将被命名为 "lt3595::"（已弃用）
- linux,default-trigger：LED 的默认触发器
参见 Documentation/devicetree/bindings/leds/common.txt
- default-state：LED 的初始状态
参见 Documentation/devicetree/bindings/leds/common.txt

如果设计中存在多个此类芯片，每个芯片都需要由其自己的设备节点处理。

示例：

```dtb
#include <dt-bindings/leds/common.h>

led-controller {
	compatible = "lltc,lt3593";
	lltc,ctrl-gpios = <&gpio 0 GPIO_ACTIVE_HIGH>;

	led {
		function = LED_FUNCTION_BACKLIGHT;
		color = <LED_COLOR_ID_WHITE>;
		default-state = "on";
	};
};
```

在这个示例中，我们定义了一个与 Linear Technologies LT3593 LED 控制器兼容的设备节点。该节点包含一个子节点 `led`，用于配置 LED 的功能、颜色和初始状态。此外，通过 `lltc,ctrl-gpios` 属性指定了与控制器 'CTRL' 引脚相连的 GPIO。
