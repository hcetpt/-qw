起重机商品系统 - EL15203000 LED 驱动器
--------------------------------------------------

此 LED 板（也称为红色 LED 板）广泛应用于由起重机商品系统生产的咖啡自动售货机。该板管理 3 个 LED，并支持为特定 LED 预定义的闪烁模式。
售卖区域 LED 用符号 'V'（十六进制码 0x56）编码，没有硬件级别的闪烁模式。
屏幕灯管 LED 包围自动售货机屏幕，并用符号 'S'（十六进制码 0x53）编码。支持呼吸式的闪烁模式。
水管 LED 用符号 'P'（十六进制码 0x50）编码，实际上由 5 个 LED 组成，在协议中像一个 LED 一样暴露。支持以下模式：
- 瀑布模式
- 反向瀑布模式
- 弹跳模式
- 反向弹跳模式

必需属性：
- compatible: "crane,el15203000"
- #address-cells: 必须为 1
- #size-cells: 必须为 0

属性规则在文档 `Documentation/devicetree/bindings/spi/spi-bus.txt` 中描述。特别是，“reg”和“spi-max-frequency”属性必须给出。
可选的 LED 子节点属性：
- function:
    查看文档 `Documentation/devicetree/bindings/leds/common.txt`
- color:
    查看文档 `Documentation/devicetree/bindings/leds/common.txt`

示例
-------

```text
#include <dt-bindings/leds/common.h>

led-controller@0 {
	compatible = "crane,el15203000";
	reg = <0>;
	spi-max-frequency = <50000>;
	#address-cells = <1>;
	#size-cells = <0>;

	/* 水管 */
	led@50 {
		reg = <0x50>;
		function = "pipe";
		color = <LED_COLOR_ID_RED>;
	};

	/* 屏幕边框 */
	led@53 {
		reg = <0x53>;
		function = "screen";
		color = <LED_COLOR_ID_RED>;
	};

	/* 售卖区域 */
	led@56 {
		reg = <0x56>;
		function = "vend";
		color = <LED_COLOR_ID_RED>;
	};
};
```
