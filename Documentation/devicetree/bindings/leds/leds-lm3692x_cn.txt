* 德州仪器 (Texas Instruments) - LM3692x 高效白光LED驱动器

LM3692x 是一款超紧凑型、高效率的白光LED驱动器，专为LCD显示屏背光设计。LM36922与LM36923的主要区别在于它们支持的LED串数量不同：LM36922支持两串LED，而LM36923支持三串LED。
必需属性：
    - compatible:
        "ti,lm36922"
        "ti,lm36923"
    - reg : I2C从机地址
    - #address-cells : 1
    - #size-cells : 0

可选属性：
    - enable-gpios : 用于启用/禁用设备的GPIO引脚
    - vled-supply : LED供电
    - ti,ovp-microvolt: 过压保护（单位微伏），可以设置为17000000、21000000、25000000或29000000。如果未指定ti,ovp-microvolt，则默认值为29000000
必需的子节点属性：
    - reg : 
        0 - 启用所有LED同步路径
        1 - 启用LED1同步
        2 - 启用LED2同步
        3 - 启用LED3同步（仅LM36923）

可选的子节点属性：
    - function : 请参阅文档/devicetree/bindings/leds/common.txt
    - color : 请参阅文档/devicetree/bindings/leds/common.txt
    - label : 请参阅文档/devicetree/bindings/leds/common.txt（已废弃）
    - linux,default-trigger : 请参阅文档/devicetree/bindings/leds/common.txt
    - led-max-microamp : 请参阅文档/devicetree/bindings/leds/common.txt

示例：

```dtb
#include <dt-bindings/leds/common.h>

led-controller@36 {
	compatible = "ti,lm3692x";
	reg = <0x36>;
	#address-cells = <1>;
	#size-cells = <0>;

	enable-gpios = <&gpio1 28 GPIO_ACTIVE_HIGH>;
	vled-supply = <&vbatt>;
	ti,ovp-microvolt = <29000000>;

	led@0 {
		reg = <0>;
		function = LED_FUNCTION_BACKLIGHT;
		color = <LED_COLOR_ID_WHITE>;
		linux,default-trigger = "backlight";
		led-max-microamp = <20000>;
	};
};
```
