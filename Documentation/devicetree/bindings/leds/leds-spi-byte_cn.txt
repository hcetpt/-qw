* 单字节 SPI LED 设备驱动程序
该驱动程序可用于具有非常简单的 SPI 协议的控制器：
- 一个 LED 由 MOSI 上的一个字节控制
- 字节的值给出了两个值之间的亮度（最低到最高）
- 不需要返回值（无需 MISO 信号）

最低和最高的亮度值取决于设备，因此也取决于兼容字符串。
根据兼容字符串，某些特殊功能（如硬件加速闪烁）也可能得到支持。
目前，该驱动程序仅支持一个 LED。LED 的属性在设备节点下的子节点中配置。

必需的属性：
- 兼容性：应为以下之一
   * "ubnt,acb-spi-led"		基于微控制器（SONiX 8F26E611LA）的设备
				例如，在Ubiquiti airCube ISP中使用

在文档 `Documentation/devicetree/bindings/spi/spi-bus.txt` 中描述了属性规则。

LED 子节点属性：
- 标签（label）：
	参见文档 `Documentation/devicetree/bindings/leds/common.txt`
- 默认状态（default-state）：
	参见文档 `Documentation/devicetree/bindings/leds/common.txt`
	仅支持 "on" 和 "off"

示例：

```
led-controller@0 {
	compatible = "ubnt,acb-spi-led";
	reg = <0>;
	spi-max-frequency = <100000>;

	led {
		label = "white:status";
		default-state = "on";
	};
};
```
