Motorola CPCAP PMIC LED 模块

此模块是 CPCAP 的一部分。关于整个芯片的更多详细信息，请参阅 `Documentation/devicetree/bindings/mfd/motorola-cpcap.txt`。
所需节点属性：
- compatible（兼容性）：应当为以下之一
   * "motorola,cpcap-led-mdl"		（主显示屏照明）
   * "motorola,cpcap-led-kl"		（键盘照明）
   * "motorola,cpcap-led-adl"		（辅助显示屏照明）
   * "motorola,cpcap-led-red"		（红色三极管）
   * "motorola,cpcap-led-green"		（绿色三极管）
   * "motorola,cpcap-led-blue"		（蓝色三极管）
   * "motorola,cpcap-led-cf"		（摄像头闪光灯）
   * "motorola,cpcap-led-bt"		（蓝牙指示灯）
   * "motorola,cpcap-led-cp"		（摄像头隐私指示灯）
- label（标签）：请参见 `Documentation/devicetree/bindings/leds/common.txt`
- vdd-supply（电源供应）：指向 LED 供电调节器的 phandle

示例：

```devicetree
&cpcap {
	cpcap_led_red: red-led {
		compatible = "motorola,cpcap-led-red";
		label = "cpcap:red";
		vdd-supply = <&sw5>;
	};
};
```

这段文档描述了如何在设备树中配置 Motorola CPCAP PMIC 中的 LED 控制器，并给出了一个具体的例子，展示了如何为红色 LED 进行配置。
