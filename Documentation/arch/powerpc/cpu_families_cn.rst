CPU 家族
========

本文件试图总结一些现存的并被 arch/powerpc 支持的不同 CPU 家族。

Book3S（即 sPAPR）
------------------

- 哈希 MMU（除了 603 和 e300）
- 径向 MMU（POWER9 及以后版本）
- 软件加载的 TLB（603 和 e300）
- 除哈希 MMU 外可选的软件加载 TLB（755、7450、e600）
- 32 位与 64 位混合：
