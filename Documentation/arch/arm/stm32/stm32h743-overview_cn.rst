STM32H743 概览
=============

简介
----

STM32H743 是一款面向多种应用的 Cortex-M7 微控制器，其特点包括：

- Cortex-M7 内核，运行频率高达 400 MHz
- 2 MB 内置闪存，1 MB 内置 RAM
- FMC 控制器用于连接 SDRAM、NOR 和 NAND 存储器
- 双模式 QSPI
- 支持 SD/MMC/SDIO
- 以太网控制器
- USB OTG FS & HS 控制器
- 支持 I2C、SPI、CAN 总线
- 多个 16 位和 32 位通用定时器
- 串行音频接口
- LCD 控制器
- HDMI-CEC
- SPDIFRX
- DFSDM
