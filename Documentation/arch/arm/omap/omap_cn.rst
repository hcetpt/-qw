OMAP 历史
==========

此文件包含了在 OMAP 上运行主线内核的文档。
=====		======================================================
KERNEL		新增依赖项
=====		======================================================
v4.3+		需要更新自定义的 `.config` 文件以确保 `CONFIG_REGULATOR_PBIAS` 被启用，这样 MMC1 才能正常工作。
v4.18+		需要更新自定义的 `.config` 文件以确保 `CONFIG_MMC_SDHCI_OMAP` 对所有 MMC 实例都被启用，以便在基于 DRA7 和 K2G 的板子上正常工作。
=====		======================================================
