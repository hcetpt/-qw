==========================
DAMON：数据访问监控器
==========================

DAMON 是一个 Linux 内核子系统，提供了一个数据访问监控框架以及基于监控结果的系统操作。DAMON 的核心监控机制（详细内容请参见 :doc:`design`）使其具备以下特点：

- *准确性*（监控输出足够用于 DRAM 级别的内存管理；不过对于 CPU 缓存级别可能不太适用），
- *轻量级*（监控开销足够低，可以在线应用），
- *可扩展性*（开销的上限与目标工作负载的大小无关，保持在常数范围内）

通过这个框架，内核可以以访问感知的方式运行系统。由于这些功能也暴露给了用户空间，拥有特殊工作负载信息的用户可以编写个性化的应用程序来更好地理解和优化他们的工作负载和系统。为了更方便地开发此类系统，DAMON 还提供了一个名为 DAMOS（基于 DAMON 的操作方案）的功能，除了监控之外。利用该功能，无论是内核空间还是用户空间中的 DAMON 用户都可以通过简单的配置而不是编写代码来进行访问感知的系统操作。
