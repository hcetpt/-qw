CPUFreq —— Linux（TM）内核中的 CPU 频率和电压调节代码
========================================================================

**作者:** Dominik Brodowski  <linux@brodo.de>

   时钟调节允许您即时更改 CPU 的时钟速度。这是一种很好的方法来节省电池电量，因为时钟速度越低，CPU 消耗的电量就越少。
