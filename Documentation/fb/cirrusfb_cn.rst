============================================
Cirrus Logic 芯片组的帧缓冲驱动
============================================

版权所有 1999 Jeff Garzik <jgarzik@pobox.com>

.. 只是一个小小的开始，欢迎贡献者参与！

支持的芯片系列：
	- SD64
	- Piccolo
	- Picasso
	- Spectrum
	- Alpine（GD-543x/4x）
	- Picasso4（GD-5446）
	- GD-5480
	- Laguna（GD-546x）

支持的总线类型：
	- PCI
	- Zorro

支持的架构：
	- i386
	- Alpha
	- PPC（Motorola Powerstack）
	- m68k（Amiga）

默认视频模式
-------------------
目前支持两个内核命令行参数：

- mode:640x480
- mode:800x600
- mode:1024x768

完整的启动视频模式（modedb）支持即将集成。
